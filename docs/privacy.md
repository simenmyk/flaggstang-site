# Personvernerklæring for Flaggstang

Sist oppdatert: 22. juni 2026

Flaggstang er ein iPhone-app som hjelper brukaren å finne norske flaggdagar, tilrådde flaggtider, vêr og lokale påminningar.

## Data appen brukar

### Posisjon

Appen kan bruke posisjonen din dersom du gir løyve til det. Posisjonen blir brukt til å hente vêr og soloppgang/solnedgang for rett område.

Koordinatar blir runda av før dei blir sende til MET Norway sine opne API. Appen brukar MET Norway for vêrdata og soltider.

Dersom du ikkje gir posisjonsløyve, brukar appen standardposisjon Oslo.

### Lokale innstillingar

Appen lagrar nokre innstillingar lokalt på eininga, til dømes valt språk, varselinnstillingar og lokal cache for soltider. Desse opplysningane blir lagra med UserDefaults/AppStorage og er berre til bruk i appen.

### Varsel

Appen kan planleggje lokale varsel dersom du gir løyve. Varsla blir planlagde på eininga. Appen sender ikkje varseldata til ein ekstern server.

## Tredjepartar

Appen hentar data frå MET Norway sine opne API for vêr og soltider. Når appen gjer dette, blir avrunda koordinatar sende til MET Norway for å kunne levere appfunksjonaliteten.

## Sporing og annonser

Appen brukar ikkje annonser, sporar ikkje brukaren på tvers av appar eller nettstader, og brukar ikkje data til reklameprofilering.

## Kontakt

For spørsmål om personvern kan du kontakte utviklaren via kontaktadressa som er oppgitt i appens User-Agent og App Store-oppføring.
